Modal

Gom.UI. Modal

new Modal(opts)

弹层底层抽象类，如果需要自定义弹出层才需要用到, 自定义一般于Modal.layout，不满足才需要用到此类 所有弹出层不可共存，但modal与toast可一起显示.

Parameters:
Name Type Description
opts object

传入的opts参数，会覆盖static默认参数

Properties
Name Type Attributes Description
type string <optional>

自定义GomUI不存在的弹出层组件时才需要指定
title string <optional>

弹层标题
content html | string <optional>

弹层的html内部
class string <optional>

弹层自定义class
mask boolean <optional>

弹层是否显示遮罩
btns object <optional>

弹出层组件时按钮

Properties
Name Type Attributes Description
yes string <optional>

确定按钮名称
no string <optional>

取消按钮名称
Source:
  • ui/ui.modal.js, line 14
To Do:
  • loading应该处理为可与其它弹出层共存
Example
<p>opts对象可传入的有如下对象，如果opts为string时，则表示为opts.content
 {  type: '',  //在具体实例中已定义，扩展时可自定义名称
           btns: {
                yes: '确定',
                no:  '取消'
            },
            title: '',
            content: '',   //content为str或html,如果为function则需要返回str或html
            class: '',
            mask: true,
            onYes: function(){},
            onNo: function(){}
            }</p>
